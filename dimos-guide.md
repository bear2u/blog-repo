# dimos 완벽 가이드

> **Physical Space를 위한 Agentive Operating System**

**dimos (Dimensional OS)**는 “로보틱스용 에이전트 네이티브 OS/SDK”를 표방하며, Python 중심의 애플리케이션/블루프린트로 다양한 하드웨어(휴머노이드/4족/드론 등)를 제어하는 워크플로우를 제공합니다. (`README.md`, `docs/usage/*`)
