## RS-SDK란?

**RS-SDK**는 RuneScape 스타일의 MMO 게임에서 봇을 개발하고 테스트할 수 있는 연구 지향 스타터 킷입니다. 복잡한 경제 RPG MMO 환경에서 봇을 구축하고 운영하며, AI 에이전트 개발 기술을 실험할 수 있습니다.

```
   ____  ____       ____  ____  _  __
  |  _ \/ ___|     / ___||  _ \| |/ /
  | |_) \___ \ ____\___ \| | | | ' /
  |  _ < ___) |_____|__) | |_| | . \
  |_| \_\____/     |____/|____/|_|\_\

  Research-oriented starter kit for runescape-style bots
```

## 프로젝트 목표

RS-SDK의 핵심 목표는 두 가지입니다:

1. **목표 지향적 프로그램 합성** - Ralph loops 등 프로그램 합성 기법을 테스트하기 위한 풍부한 환경 제공
2. **에이전트 연구** - 에이전트 간 협력과 경쟁에 대한 연구 촉진

## LostCity 기반

RS-SDK는 **LostCity 엔진**의 포크입니다. LostCity는 2004년 RuneScape를 에뮬레이션하는 놀라운 오픈소스 프로젝트입니다.

```
┌─────────────────────────────────────────────────────┐
│                    RS-SDK                           │
├─────────────────────────────────────────────────────┤
│                                                     │
│   LostCity Engine (2004scape)                       │
│   ├── 서버 에뮬레이터                               │
│   ├── 웹 클라이언트                                 │
│   └── 게임 콘텐츠 (맵, 스크립트, 에셋)             │
│                                                     │
│   + RS-SDK 추가                                     │
│   ├── TypeScript Bot SDK                            │
│   ├── Gateway 서비스                                │
│   ├── MCP 통합                                      │
│   └── Bot 자동화 도구                              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## 주요 구성 요소

### 1. 게임 서버 (Engine)

LostCity 기반의 게임 서버로, 월드 상태, 플레이어, NPC, 게임 로직을 처리합니다.

### 2. 웹 클라이언트 (WebClient)

브라우저에서 실행되는 TypeScript 기반 게임 클라이언트입니다.

### 3. Bot 클라이언트 (BotClient)

SDK와 통신하도록 확장된 특수 웹 클라이언트입니다.

### 4. Gateway

Bot 클라이언트와 SDK 에이전트 간의 WebSocket 통신을 중계합니다.

### 5. BotSDK

TypeScript로 봇 자동화 스크립트를 작성할 수 있는 라이브러리입니다.

## 게임플레이 수정 사항

원본 게임과 달리 RS-SDK 서버는 봇 개발을 위해 몇 가지 수정이 적용되어 있습니다:

| 수정 사항 | 설명 |
|-----------|------|
| **빠른 레벨링** | XP 커브가 가속되고 덜 가파름 |
| **무한 런 에너지** | 에너지가 소모되지 않음 |
| **랜덤 이벤트 비활성화** | 안티봇 랜덤 이벤트 없음 |

## Task Length Distribution

다양한 난이도와 길이의 작업을 자동화할 수 있습니다:

- **단순 작업**: 나무 베기, 물고기 잡기, 광물 채굴
- **중간 작업**: 전투 훈련, 제작, 요리
- **복잡한 작업**: 뱅킹 루프, 여러 위치 이동, 퀘스트

## 면책 조항

- 무료, 오픈소스, 커뮤니티 운영 프로젝트입니다
- 목표는 엄격하게 교육과 과학 연구입니다
- Jagex Ltd.와 공식적으로 승인받거나 통신한 적이 없습니다
- 여기서 Old School RuneScape를 플레이하거나 골드를 구매할 수 없습니다
- **여기서 개발한 봇은 공식 게임 서버에서 작동하지 않습니다**
