## 설치 개요(README 기준)

설치 방식이 플랫폼별로 다릅니다.

- Claude Code / Cursor: 플러그인(마켓플레이스) 방식
- Codex / OpenCode: INSTALL 문서를 따라 수동 설치

---

## Codex 설치(README 안내)

Codex에게 아래처럼 요청해 설치 문서를 따라가도록 합니다.

```
Fetch and follow instructions from https://raw.githubusercontent.com/obra/superpowers/refs/heads/main/.codex/INSTALL.md
```
