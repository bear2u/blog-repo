# GitNexus 완벽 가이드

> **Building nervous system for agent context.** (`README.md`)

**abhigyanpatwari/GitNexus**는 코드베이스를 지식 그래프(의존성/콜체인/클러스터/실행 플로우)로 인덱싱하고, CLI + MCP(Web UI 포함)로 AI 에이전트가 구조를 “놓치지 않게” 돕는 도구를 목표로 합니다. (`README.md`, `gitnexus/package.json`)
